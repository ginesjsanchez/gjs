## PYZ maven extension 

 Project PYZ Maven Extension 
 
## Description

Project plugin in java language.

Define the pyz packaging for maven projects 


## Requirements

* Maven 3.9.x


## Use

<packaging>pyz</packaging>

<build>
	<extensions>
		<extension>
			<groupId>org.gjs.maven.extensions</groupId>
			<artifactId>pyz-maven-extension</artifactId>
			<version>1.0.0-RC.1</version>
		</extension>
	</extensions>
</build>

    mvn clean install
